INTRODUCTION
------------

The Map Taxonomy Terms module provides a mechanism to map (hide/show) taxonomy vocabularies with terms
as per the requirement, without actually deleting them.


INSTALLATION
------------

Extract module at drupal/modules/contrib directory and enable it from browser
by going in this path /admin/modules.


CONFIGURATION
-------------

1. Configure at admin/config/web services/Configure Taxonomy Mapping.
2. Select the terms from the respective vocabularies.
3. Check the 'Enable Mapping' checkbox.
4. Save Configuration.
